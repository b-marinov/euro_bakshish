## For Drivers ğŸš™

### 1. Create Your Account (2 minutes)
1. Visit http://localhost (or your deployment URL)
2. Click "Register"
3. Fill in:
   - Username, Email, Password
   - First & Last Name
   - Select "Driver" as User Type
4. Click "Register"

### 2. Complete Your Driver Profile (2 minutes)
1. Login with your credentials
2. Go to "Profile"
3. Add:
   - **Vehicle Details**: Make, model, year, color, license plate
   - **License Info**: Driver's license number, expiration date
   - **Availability**: Set to "Available"
4. Save changes

### 3. Accept Your First Ride (1 minute)
1. View "Available Trips" from dashboard
2. Browse pending requests
3. Click on a trip to see details
4. Click "Accept Trip" if interested
5. Click "Start Trip" when you pick up the passenger
6. Click "Complete Trip" when you reach the destination
7. Rate your passenger!

## Common Actions

### Cancel a Trip
**Passengers & Drivers:**
1. Go to "Active Trips"
2. Select the trip
3. Click "Cancel Trip"
4. Confirm cancellation

### Rate Someone
1. Go to "Pending Reviews" after trip completion
2. Select the completed trip
3. Rate 1-5 stars for each category
4. Add optional comments
5. Submit

### Update Availability (Drivers)
1. Go to "Profile"
2. Toggle "Availability Status"
3. Save changes

### View Your Ratings
1. Click "My Ratings" in navigation
2. See your average rating
3. View individual reviews
